Снижение размерности с помощью нейронных сетей.¶

Autoencoder.¶

Здесь мы решим задачу снижения размерности с помощью аппарата нейронных сетей, который получил наименование autoencoder. Автоэнкодеры — это искусственные нейронные сети, которые позволяют построить так называемые коды. Коды обычно имеют меньшую размерность, чем входные данные. Это делает их популярным методом уменьшения размерности Автоэнкодеры могут помочь предварительно обучить нижние уровни глубоких нейронных сетей. сети, особенно когда данные имеют ограниченное количество меток Автоэнкодеры можно использовать как генеративные модели, т. е. они могут генерировать новые данные, аналогичные обучающим данным. Наконец, автоэнкодеры стали популярным методом обнаружения аномалий. Автоэнкодер состоит из 2 частей:

  • Кодер (сеть распознавания) преобразует входные данные во внутренний представление
  • Декодер (генеративная сеть) преобразует внутреннее представление в выходы

Поскольку автоэнкодер тренируется, делая выходные данные равными входным, количество входы и выходы должны быть равны Выходы автоэнкодера также называются реконструкциями или потерями. Автоэнкодер обычно неполный, т.е. внутреннее представление имеет меньшую размерность, чем входы. Это предотвращает тривиальные решения, когда входы просто копируются в выходы Автоэнкодер может быть неполным по-разному:

  • Слой внутренних представлений меньше, чем количество входов

  • Внутренний слой представления имеет много узлов, но меньшее количество соединений между узлами (разреженный автоэнкодер)

У автоэнкодера есть как кодировщик, так и декодер, содержащий несколько слоев нейронов называется многоуровневым автоэнкодером или глубоким автоэнкодером. Обычно стекированные автоэнкодеры имеют одинаковые размеры слоев, которые симметрично расположены относительно самого узкого места. Кроме того, веса слоя декодирования часто равны транспонированному веса симметричного ему слоя кодирования. Это называется завязывать веса. Связывание весов уменьшает количество весов в сети и ускоряет обучение. Мотивация за дополнительными слоями такая же, как и за глубокими нейронными. сети: более сложный предмет для изучения с меньшим числом параметров. Однако добавление слишком большого количества слоев может привести к тривиальным решениям или потере эффективности из-за недостаточной полноты

from tensorflow.keras.layers import Dense, Input

In [110]:
encoding_dim = 3
# encode:
input_layer = Input(shape=(RatesData.shape[1],))
encoded_layer = Dense(encoding_dim, use_bias=False, activation=None)
encoded = encoded_layer(input_layer)

# decode:
input_encoded = Input((encoding_dim,))
decoded_layer = Dense(RatesData.shape[1], use_bias=False, activation=None)
decoded = decoded_layer(input_encoded)

# models:
encoder = Model(input_layer, encoded, name="encoder")
decoder = Model(input_encoded, decoded, name="decoder")
autoencoder = Model(input_layer, decoder(encoder(input_layer)), name="autoencoder")

lr = 0.0005 # learning rate, defaults to 0.001
autoencoder.compile(optimizer=Adam(lr), loss='mse')  

autoencoder.summary()

def defineOrder(pca_factors,rot_factors):
    pca_cols = pca_factors.columns
    rot_cols = rot_factors.columns
    ncol = len(pca_cols)
    mylist = []
    alllist = []
    i = 0

    for pca_col in pca_cols:
        pcaf =pca_factors[pca_col]
        j = 0
        for rot_col in rot_cols:
            rotf = rot_factors[rot_col]
            cor = np.corrcoef(pcaf,rotf)[0,1]
            print(i,j,abs(cor))
            mylist = []
            mylist.append(i)
            mylist.append(j)
            mylist.append(abs(cor))
            mylist.append(np.sign(cor))
            alllist.append(mylist)
            j = j+1
        i = i +1
    df = pd.DataFrame(alllist,columns = ['col1','col2','value','sign'])
    df = df.sort_values(by=['value'], ascending=False)
    df = df[:ncol]
    df = df.sort_values(by='col1')
    return df
